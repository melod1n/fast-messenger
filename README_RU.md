## О проекте

Fast Messenger - современный Android-клиент, ориентированный на основные сценарии общения в VK. Проект сочетает полностью написанный на Jetpack Compose интерфейс, многомодульную архитектуру с разделением по фичам, локальное хранение данных, постраничную загрузку и обновления в реальном времени через VK Long Poll.

Это также личный инженерный проект для изучения и применения современных практик Android-разработки в нетривиальном продукте с авторизацией, диалогами, различными типами содержимого сообщений, обработкой медиа, фоновыми обновлениями и сложным состоянием UI.

## Архитектура

Fast Messenger использует многомодульную структуру с разделением по фичам. Состояние UI управляется через `ViewModel` и публикуется с помощью Kotlin `StateFlow`. Бизнес-логика вынесена в доменные сценарии использования, а сетевое взаимодействие, хранение данных, настройки, модели и переиспользуемые UI-компоненты находятся в отдельных core-модулях.

```text
app
├── feature
│   ├── auth
│   ├── chatmaterials
│   ├── convos
│   ├── createchat
│   ├── friends
│   ├── languagepicker
│   ├── messageshistory
│   ├── photoviewer
│   ├── profile
│   └── settings
├── core
│   ├── common
│   ├── data
│   ├── database
│   ├── datastore
│   ├── domain
│   ├── logger
│   ├── model
│   ├── network
│   ├── presentation
│   └── ui
└── build-logic
```

Проект также содержит собственные convention plugins для Gradle, централизованный version catalog, проверку графа зависимостей между модулями и анализ стабильности Compose.

## Сборка

Клонируйте репозиторий:

```bash
git clone https://github.com/melod1n/fast-messenger.git
cd fast-messenger
```

Соберите debug APK:

```bash
./gradlew assembleDebug
```

Готовый APK будет находиться в каталоге:

```text
app/build/outputs/apk/debug/
```

Для release-сборки требуется настроенная подпись. Готовый к установке APK можно скачать из последнего релиза на GitHub.

## Установка

Скачайте последнюю доступную версию APK на странице [Релизов](https://github.com/melod1n/fast-messenger/releases).

Поскольку приложение распространяется вне Google Play, Android может запросить разрешение на установку приложений из браузера или файлового менеджера.

## Статус проекта

Fast Messenger находится в разработке. Основные функционал с диалогами и сообщениями уже реализован, однако набор возможностей пока не соответствует (и, возможно, никогда не будет соответствовать) официальному клиенту VK.

Сообщения об ошибках и целевые pull request'ы принимаются через [GitHub Issues](https://github.com/melod1n/fast-messenger/issues).
